# Product Hunt Launch Pack

## Name

LittleCoder

## Tagline

Local AI coding worker with visible execution and a safe factory workspace

## Short Description

LittleCoder lets you watch an AI coding worker build software live in Studio while everything stays inside a protected factory workspace.

## Full Description

LittleCoder is a local AI coding worker built for people who want visibility, simplicity, and control.

Instead of hiding execution behind a black box, LittleCoder shows what the Worker is doing in real time:
- files appear in the factory workspace
- code shows up live in Studio
- terminal output streams as commands run

It is local-first, Studio-first, and designed as a focused product rather than a sprawling AI framework.

## Launch Highlights

- local Ollama support
- safe factory workspace isolation
- Studio-first task control
- visible file and terminal execution
- optional Telegram control with narrated progress

## Maker Comment

I built LittleCoder because I wanted AI coding to feel understandable instead of magical.

The goal was not to create another agent framework.
The goal was to build a coding worker you can actually watch, trust, and control.
