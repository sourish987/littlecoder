# LittleCoder v0.1.0

Local AI Coding Worker with Visible Execution.

LittleCoder is a Studio-first AI coding worker designed to make software creation understandable, visible, and local-first.

## Highlights

- Studio-first workflow
- safe factory workspace isolation
- local Ollama-based planning
- visible file creation and terminal execution
- optional Telegram control
- narrated remote execution progress

## What You Can Do

- install with `npm install`
- run guided setup with `npm run setup`
- start the Worker with `npm start`
- create projects inside a safe factory workspace
- watch execution happen live in Studio

## Suggested First Tasks

- `create a simple website`
- `create a todo website`
- `create a personal portfolio`
- `create a calculator webpage`
- `create a landing page`

## Why LittleCoder

Most AI coding tools feel like a black box.

LittleCoder is built to feel visible and trustworthy:
- one Worker
- one Studio
- one safe factory workspace
- local-first by default

## Notes

- Telegram is optional
- Studio is the primary control surface
- the product is intentionally simple and focused
