# FAQ

## Does LittleCoder require Telegram?

No. Studio is the primary interface. Telegram is optional.

## Does LittleCoder work offline?

It is designed for local-first use with Ollama. Internet access is not required for core operation once dependencies are installed.

## Where does it write files?

Inside the factory workspace only.

## Can it write outside the factory?

No. The product is designed around a safe factory workspace model.

## Is this a multi-agent framework?

No. LittleCoder is a single AI coding worker product.

## What should I try first?

Use Studio and start with:
- `create a simple website`
- `create a todo website`
- `create a personal portfolio`
